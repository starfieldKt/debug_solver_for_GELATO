# debug_solver_for_GELATO

GELATOの検証用に作ったソルバーです。
任意の時系列の水位と流速の係数を入力するとそれっぽい結果ができます。

基本の流向はi方向j方向にちょうど1セル移動するように作ってますので係数で調整してください。
